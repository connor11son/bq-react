Seam Carving Detector

Here is the forensic tool to flag images manipulated using a technique called Seam Carving. Seam carving can be used for object removal, object dislocation, and content-aware image retargeting.

Input

.png or .jpeg image

Output

.png image with overall image level manipulation score, in addition to, Seam Prediction mask indicating the regions of the image that are prone to seam removal (in red) and seam insertion (in green).
